# OrangeHRM Automation Framework

## Overview

This project is a Selenium Automation Framework developed using Java, Selenium WebDriver, TestNG, Maven, and the Page Object Model (POM).

The framework automates the OrangeHRM login functionality and follows industry-standard automation framework design with reusable page classes and configuration management.

## Automated Test Scenario

### Login Test

- Launch Chrome Browser
- Open OrangeHRM Application
- Enter Username
- Enter Password
- Click Login
- Verify Dashboard Page
- Close Browser
